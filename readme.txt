MagPie allows you to import products from/to CSV file and image for WP e-Commerce.

== Description ==

MagPie allows you to import products from CSV file and image. You have your products in a spreadsheet or database and want to be able to export a CSV file and import your products automatically into WP e-Commerce. MagPie is for you.

== Installation ==

1. Download and unzip to your plugin folder.
2. Activate the plugin from the Plugins section of your dashboard.
